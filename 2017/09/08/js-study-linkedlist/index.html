Array vs Linked List

Array vs Linked List

  • Size => Fixed vs Dynamic
  • Insert => Hard vs Easy
  • Deletion => Hard vs Easy
  • Random Access(원하는 곳에 한번에 접근할 수 있다) => Allowed vs Not allowed
  • Extra memory space => doesn’t need vs required
