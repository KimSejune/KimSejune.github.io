Express

  • Node.js 생태계에서 가장 널리 쓰이는 웹 프레임워크
  • 내장하고 있는 기능은 매우 적으나, 미들웨어를 주입하는 방식으로 기능을 확장하는 생태계를 가지고 있음
  • 공식 매뉴얼 한국어 번역

Express 앱의 기본 구조

Routing

Request 객체

req.body

  • requset.body를 적절한 형태의 자바스크립트 객체로 변환하여 이곳에 저장 (body-parser 미들웨어에 의해 처리됨)

req.ip

  • 요청한 쪽의 IP

req.params

  • route parameter

req.query

  • query string이 객체로 저장됨

Response 객체

res.status(...)

  • 응답의 상태 코드를 지정하는 메소드

res.append(...)

  • 응답의 헤더를 지정하는 메소드

res.send(...)

  • 응답의 바디를 지정하는 메소드 인자가 텍스트면 text/html, 객체면 application/json타입으로 응답

숫자를 send에 넣으면 error가 발생한다.
